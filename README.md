# お店の物語アートメーカー（つるるんとん デモ版）

店舗の世界観を、**1話完結の絵本ストーリー**と**画像設計**（キービジュアル・店内アート・ハガキ）に変換する Web アプリのデモです。
AI学校の課題提出用に、**画像生成APIが未接続でも一通り動く**ように作っています。

- 技術：Next.js（App Router）＋ TypeScript ＋ Tailwind CSS
- データ保存：localStorage（ログイン・課金・複数店舗管理なし）
- スマホ／iPad ファーストのレスポンシブUI

---

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開きます。
初回は「大きな荷物を背負ったくま君」のサンプル作品が自動で入っています。

## 設計の考え方

### 画像生成の「失敗」を減らす仕組み

- **背景は自由生成させず、背景プレートとして固定**（`/backgrounds`）
- 椅子・鏡・施術スペースなどの**固定オブジェクトを属性として登録**
- 顔剃り・シャンプー椅子・鏡の反射は**高難度シーン**として扱う
- 高難度シーンには**安全構図テンプレート**を用意（`/templates`、自動適用）
- 生成前に**リスクチェック**を表示（`/image-design`・`/risk-check`）
- **画像プロンプトを自動生成**（ユーザーはゼロから書かない）
- **修正はボタン式**（`/image-fix`）。押すと指示文が自動で出る
- 採用画像・不採用画像・採用構図を**保存**できる（`/works`）

### 表現チェック

医療・健康・美容の誇大／断定表現（治る・必ず改善・絶対 など）を検出し、
やわらかい言い換え（整う・変化を感じる方もいます など）を提案します。
（`lib/expressionCheck.ts`）

## ディレクトリ構成（差し替えポイント）

```
app/                 … 12画面（App Router）
components/           … AppShell（ナビ）・共通UI
lib/
  types.ts           … 全データ型
  seed.ts            … つるるんとん固定設定（★店舗ごとに差し替えれば汎用化）
  store.tsx          … localStorage 状態管理（React Context）
  generate.ts        … ★物語生成（テンプレ）。AI接続時はここだけ差し替え
  imagePrompt.ts     … ★画像プロンプト生成 ＋ generateImage()（画像APIの唯一の窓口）
  expressionCheck.ts … 表現チェック
  fixButtons.ts      … 修正ボタン定義
  outputs.ts         … 出力サイズ（店内アート・ハガキ）
```

### 物語の生成（2モード）

1話作成画面で「生成方法」を選べます。

- **🆓 無料テンプレ**（`lib/generate.ts`）：API不要。会話駆動・段階的な告白・終盤の反転・
  比喩タイトル・締めの一文・静寂/五感/反復の技法を入れた“強化テンプレ”。しっかり読ませる＝約2,000字。
- **✨ AIで本格生成（BYOK）**（`lib/ai.ts`）：利用者自身の Anthropic APIキーを使い、
  ブラウザから直接 Anthropic API を呼んで、人が書いた絵本に近い感動を生成。
  既定モデルは `claude-opus-4-8`。キーは端末のlocalStorageのみに保存（サーバーには送らない）。
  運営に継続課金が発生しない設計。

### 画像生成APIのつなぎ方（将来）

- **画像**：`lib/imagePrompt.ts` の `generateImage(prompt)` の中身に
  Nano Banana Pro などの呼び出しを実装し、`IMAGE_API_ENABLED = true` にする。
  返ってきた画像URLは `ImageDesignItem.imageUrl` に入り、プレースホルダーが実画像に変わる。
